Embroidery Pattern Designer - Help

Step-By-Step Instructions

  1. Check "Add Mode" and then click on the canvas at the desired location to add each new stitch.
  2. Add new layers of stitches by clicking "Add Layer".
  3. Uncheck "Add Mode" and select any stitches by clicking or dragging a square.
  4. Move selected stitches with drag and drop.
  5. Delete selected stitches stitch by pressing "delete" or clicking "Delete Selected".
  6. Save the whole pattern in .exp format by checking "More Options" and clicking "Save to EXP".

Helpful Tips

  • Add as many stitches as you want to. That is how you draw your pattern.
  • Add multiple layers to make discrete figures in the pattern.
  • Change colors of a selected layer by clicking the color box next to its name.
  • Randomize colors of all layers in your pattern with the "New Colors" button.
  • Use drag and drop stitch movement or deletion to fix pattern problems without starting over.
  • Use the Undo and Redo buttons to recover changes
  • Want more information about the selected a stitch? Check "Show Stitch Info".
  • While Add mode is unchecked, select all the stitches in a layer by clicking the layer name.
